About the Project

This project was all about refactoring code into functions to make the code cleaner. I used refactoring to make a function called draw_house that used functions like jump_to, draw_rectangle, draw_triangle, and draw_square to draw different parts of the house.

Sample Code: draw_house

This function creates a red house with a roof and blue window. The parameters allow for flexible positioning and scaling:

• t: The turtle graphics object for drawing
• x, y: Starting position coordinates
• s: Side length of objects
• l: Length of a rectangle
• c: Color of the objects

def draw_house(t, x1, x2, x3, y1, y2, 3, s1, s2, s3, l1, c1, c2, c3): """code that can draw a house/cabin of any size, color, and position""" jump_to(t, x1, y1) draw_rectangle(t, s1, l1, c1) jump_to(t, x2, y2) draw_triangle(t, s2, c2) jump_to(t, x3, y3) draw_square(t, s3, c3)
